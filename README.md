# HR-Analytics-Dashboard📊
This project provides an analytical overview of HR data using Power BI. It explores key HR metrics such as employee satisfaction, turnover rate, salary trends, departmental distribution, and more to derive meaningful insights and support data-driven decision-making in Human Resources.

📁 Project Structure
├── HR_Analytics.csv               # Raw HR data used for analysis
├── HR Data Analysis.pbix         # Power BI report file (editable)
├── HR Analytics Dashboard.png    # Screenshots of visual reports
└── README.md                     # Project documentation
📌 Key Features
Interactive Dashboards: Built with Power BI, allowing dynamic filtering and deep dives.
KPI Metrics: Includes employee attrition, average working years, job satisfaction, and more.
Visual Insights: Graphs and charts for easy interpretation of trends and patterns.
Data-Driven HR Decisions: Supports evidence-based HR strategies.
📊 Dashboards & Visuals
The report contains the following main visuals:

Attrition by Department and Job Role
Employee Distribution by Education and Gender
Monthly Income by Job Role
Satisfaction Metrics and Work-Life Balance
Heatmaps for Job Involvement and Performance
Screenshots and PDF exports are provided for preview without requiring Power BI.

🧮 Data Source
File: HR_Analytics.csv
Columns: Age, Job Role, Monthly Income, Department, Attrition, Education, Gender, Years at Company, etc.
Note: This is dummy HR data commonly used for HR analytics projects.
